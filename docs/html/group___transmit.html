uint8_t RAIL_TxDataLoad ( RAIL_TxData_t *  txData)

Load payload to send.

Parameters
[in]txDataPointer to a RAIL_TxData_t structure which defines the payload bytes and length to transmit. If the fields are configured for fixed length.
Returns
Returns 0 on success and an error code on failure

This function may overwrite current TX data held by RAIL, and should not be called repetitively or during TX. The recommended way to use this is to call RAIL_TxDataLoad and RAIL_TxStart almost immediately in succession.

int32_t RAIL_TxPowerGet ( void  )

Get the radio transmit power level.

Returns
TX Power Level defined in deci dBm (0.0 dBm)

This will return what the power output was actually set to, not just the value passed into RAIL_TxPowerSet.

int32_t RAIL_TxPowerSet ( int32_t  powerLevel)

Set the radio transmit power level.

Parameters
[in]powerLevelTX Power Level defined in deci dBm (0.0 dBm)
Returns
TX Power Level in deci dBm (0.0 dBm)

Not all values of powerLevel are achievable, but this function will set the power output to be close to the given powerLevel, and return the value that was set as the power.

uint8_t RAIL_TxStart ( uint8_t  channel,
RAIL_PreTxOp_t  preTxOp,
void *  preTxOpParams 
)

Non-blocking Transmit.

Parameters
[in]channelDefine the channel to transmit on.
[in]preTxOpFunction to use for any pre-transmit operation (e.g. for scheduled transmit, CSMA, LBT, etc.), or NULL for an immediate transmit.
[in]preTxOpParamsPointer to the pre-transmit operation's configuration parameters, or NULL if none.
Returns
Returns 0 on successfully initiating the transmit process, or an error code on failure. If successfully initiated, transmit completion or failure will be reported by later callbacks RAILCb_TxPacketSent() (success) or RAILCb_TxRadioStatus() (failure).

Begins transmission of the payload previously loaded via RAIL_TxDataLoad(). Return error if currently transmitting or receiving.
